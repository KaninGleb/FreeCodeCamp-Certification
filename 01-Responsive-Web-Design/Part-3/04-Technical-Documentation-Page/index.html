Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language.

Inside a host environment (for example, a web browser), JavaScript can be connected to the objects of its environment to provide programmatic control over them.

function greetMe(yourName) { alert("Hello " + yourName); }

It contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements.

JavaScript and Java

JavaScript and Java are similar in some ways but fundamentally different in others.

The JavaScript language resembles Java but does not have Java's static typing and strong type checking.

// Java requires type declaration int x = 5; // JavaScript is loosely typed let x = 5;

JavaScript follows most Java expression syntax, naming conventions and basic control-flow constructs which was the reason why it was renamed from LiveScript to JavaScript.

Hello world

To get started with writing JavaScript, open the Scratchpad and write your first "Hello world" JavaScript code:

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");

Select the code in the pad and hit Ctrl+R to watch it unfold in your browser!

  • First, create a function that takes a name parameter.
  • Then, call the function with your desired name.
Variables

You use variables as symbolic names for values in your application.

JavaScript variables are declared in three ways:

  • Using var (older syntax)
  • Using let (block-scoped)
  • Using const (constant, cannot be reassigned)
var x = 42; // Function-scoped let y = 13; // Block-scoped const z = 3.14; // Constant

Always prefer const over let, and let over var when declaring variables.
